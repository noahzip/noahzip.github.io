1. 引言

Rust 提供了多种数据类型，用来处理不同类型的数据。Rust 是静态类型语言，意味着在编译时每个变量的类型就会被确定。理解这些数据类型对于编写高效且安全的程序至关重要。

2. 基本数据类型

Rust 的基本数据类型可以分为标量类型和复合类型。

2.1 标量类型

标量类型表示单一值。Rust 提供了以下三种标量类型：

  • 整型（Integer）
    整型用于表示没有小数部分的数字，分为有符号和无符号整数。

    let x: i32 = 42;  // 有符号整型，默认类型为 i32
let y: u8 = 255;   // 无符号整型

    Rust 的整型包括：

    • i8, i16, i32, i64, i128（有符号整型）
    • u8, u16, u32, u64, u128（无符号整型）
    • isize 和 usize 根据目标平台的架构而定

  • 浮点型（Floating-point）
    浮点数用于表示有小数部分的数字，Rust 提供了两种浮点类型：

    let x: f32 = 3.14;  // 单精度浮点数
let y: f64 = 3.1415926535;  // 双精度浮点数，默认类型是 f64

  • 布尔型（Boolean）
    布尔类型用于表示 true 或 false 两个值。

    let t = true;
let f: bool = false;

  • 字符型（Character）
    Rust 使用 Unicode 字符表示字符，因此字符类型（char）表示一个字符（如字母、符号或 emoji）。

    let c: char = 'z';
let heart_eyed_cat: char = '😻';

2.2 复合类型

复合类型是可以将多个值组合在一起的类型。Rust 有两种复合类型：

  • 元组（Tuple）
    元组是将多个值组合在一起，可以是不同类型的值。元组使用括号 () 来表示。

    let tup: (i32, f64, char) = (500, 6.4, 'y');
let (x, y, z) = tup;  // 解构元组

  • 数组（Array）
    数组是相同类型元素的集合，数组的大小在编译时必须确定。数组使用方括号 [] 来表示。

    let a = [1, 2, 3, 4, 5];  // 数组的类型为 [i32; 5]
let first = a[0];         // 访问数组的第一个元素

3. 类型别名

Rust 允许创建类型别名。类型别名并不会创建新的类型，它只是给现有类型起一个别名。

type Kilometers = i32;
let x: Kilometers = 100;

4. 强制类型转换

Rust 不支持隐式类型转换，但它提供了显式类型转换。你可以使用 as 关键字来进行转换。

let x = 42;
let y: f64 = x as f64;  // 将整数转换为浮点数

5. 类型推导

Rust 支持类型推导，意味着 Rust 编译器会根据上下文推导出变量的类型。

let x = 42;  // Rust 推导 x 为 i32 类型

但是，并不是所有情况下 Rust 都能推导出类型，特别是当你没有给出足够信息时，你可能需要显式指定类型。

6. 小结

  • Rust 中的基本数据类型包括标量类型（如整型、浮点型、布尔型、字符型）和复合类型（如元组和数组）。
  • 通过类型推导，Rust 可以自动推断变量的类型，但你也可以显式地指定类型。
  • 类型别名允许给现有类型定义一个新的名称。
  • Rust 不支持隐式类型转换，而是要求显式地进行类型转换。

了解并掌握 Rust 的数据类型是写出高效且安全代码的基础。

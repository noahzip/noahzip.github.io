Rust 控制流

Rust 提供了多种控制流结构，包括 if 表达式、循环（loop、while、for）以及 match 表达式。这些结构允许你根据不同的条件执行代码。

1. if 表达式

Rust 的 if 语句不仅仅是条件判断语句，它也可以作为表达式使用，这意味着 if 的结果可以被赋值给一个变量。

示例：

let number = 6;

if number % 4 == 0 {
    println!("Number is divisible by 4");
} else if number % 3 == 0 {
    println!("Number is divisible by 3");
} else {
    println!("Number is not divisible by 4 or 3");
}

if 语句还可以用于返回值：

let condition = true;
let number = if condition { 5 } else { 6 };  // 使用 if 表达式返回值
println!("The value of number is: {}", number);

2. loop 循环

loop 是 Rust 中最基本的循环形式，它会一直执行，直到显式地被打断。

示例：

loop {
    println!("This will loop forever!");
}

要打破无限循环，可以使用 break 语句。

示例：

let mut counter = 0;

loop {
    counter += 1;
    if counter == 10 {
        break;  // 退出循环
    }
}

2.1 嵌套循环

如果存在嵌套循环，break 和 continue 应用于此时最内层的循环。你可以选择在一个循环上指定一个循环标签（loop label），然后将标签与 break 或 continue 一起使用，使这些关键字应用于已标记的循环而不是最内层的循环。

示例：

fn main() {
    let mut count = 0;  
    'counting_up: loop {  // 外层循环，标记为 'counting_up 用于跳出循环
        println!("count = {}", count);  
        let mut remaining = 10;  // 初始化剩余次数为 10

        loop {  // 内层循环
            println!("remaining = {}", remaining);  // 打印当前的 remaining 值
            if remaining == 9 {  // 如果 remaining 值为 9，则跳出内层循环
                break;
            }
            if count == 2 {  // 如果 count 值为 2，跳出外层循环
                break 'counting_up;
            }
            remaining -= 1;  
        }

        count += 1;  
    }
    println!("End count = {}", count);  
}

3. while 循环

while 循环基于某个条件判断是否继续执行，直到条件为 false 时停止。

示例：

let mut number = 3;

while number != 0 {
    println!("{}!", number);
    number -= 1;
}

println!("LIFTOFF!!!");

4. for 循环

for 循环是迭代集合的常见方式。在 Rust 中，for 循环用于遍历一个集合（如数组、范围等）。

示例：

let a = [10, 20, 30, 40, 50];

for element in a.iter() {
    println!("The value is: {}", element);
}

Rust 也提供了非常灵活的范围类型，可以轻松创建一个循环的范围。

示例：

for number in 1..4 {  // 注意这个范围是半开区间
    println!("Number: {}", number);
}

5. match 表达式

match 表达式是 Rust 中非常强大和灵活的控制流结构，用来进行模式匹配。它类似于其他语言中的 switch 语句，但更加复杂和强大。

示例：

let number = 13;

match number {
    1 => println!("One"),
    2 => println!("Two"),
    3..=12 => println!("Between 3 and 12"),  // 匹配 3 到 12 的范围
    13 | 14 => println!("Thirteen or Fourteen"),  // 使用 `|` 来匹配多个值
    _ if number % 2 == 0 => println!("Even number"),  // 使用 `if` 来进一步判断
    _ => println!("Anything else"),  // 默认匹配
}

match 表达式可以匹配字面值、范围、以及更多复杂的模式。

6. 控制流的表达式

在 Rust 中，if、loop、while、for、match 都是表达式，这意味着它们的结果可以直接用于赋值或其他操作。

7. 总结

  • Rust 提供了 if、loop、while、for 和 match 等控制流结构。
  • if 是条件判断结构，可以作为表达式返回值。
  • loop 是无限循环，需使用 break 来退出。
  • while 循环基于条件判断，适合执行条件限定的操作。
  • for 循环通常用于遍历集合。
  • match 是一种强大的模式匹配表达式，可以处理不同的条件并进行复杂的匹配。

通过这些控制流语句，你可以在 Rust 中高效地管理代码的执行流。

创建一个命令行工具，能够读取指定文件并搜索用户输入的字符串。我们将结合文件I/O、命令行参数、错误处理、环境变量等知识，实现一个高效、健壮的命令行工具。

代码实现

lib.rs 代码

use std::fs;
use std::error::Error;
use std::env;

// Config 结构体存储了程序需要的配置信息
pub struct Config {
    pub search_str: String,  // 搜索的字符串
    pub file_name: String,   // 要搜索的文件名
    pub case_insensitive: bool, // 是否忽略大小写
}

// Config 的实现部分
impl Config {
    // 创建一个新的 Config 实例，从命令行参数中获取配置
    pub fn new(mut args: std::env::Args) -> Result<Config, &'static str> {
        // 跳过第一个参数，即程序的路径
        args.next();

        // 获取搜索字符串，如果没有提供，返回错误
        let _search_str = match args.next() {
            Some(arg) => arg,
            None => return Err("Don't get a search string"),  // 没有提供搜索字符串
        };

        // 获取文件名，如果没有提供，返回错误
        let _file_name = match args.next() {
            Some(arg) => arg,
            None => return Err("Don't get a file name"),  // 没有提供文件名
        };

        // 创建 Config 实例，并读取环境变量 "RIPGREP_CASE_INSENSITIVE" 判断是否忽略大小写
        Ok(Config {
            search_str: _search_str,
            file_name: _file_name,
            case_insensitive: env::var("RIPGREP_CASE_INSENSITIVE")
                .unwrap_or_default() == "1", // 如果环境变量存在并且等于 "1"，则设置为 true
        })
    }
}

// 运行程序，执行文件内容的搜索操作
pub fn run(config: Config) -> Result<(), Box<dyn Error>> {
    // 读取文件内容
    let content = fs::read_to_string(&config.file_name)?;
    let result = if config.case_insensitive {
        search_case_insensitive(config.search_str, &content)  // 不区分大小写的搜索
    } else {
        search(config.search_str, &content)  // 区分大小写的搜索
    };

    // 打印搜索结果
    for line in result {
        println!("{}", line);
    }

    Ok(())
}

// 区分大小写的搜索函数，返回匹配行的引用
pub fn search<'a>(search_str: String, content: &'a String) -> Vec<&'a str> {
    // 使用迭代器和闭包优化搜索操作
    content
        .lines() // 将内容按行分割
        .filter(|line| line.contains(&search_str)) // 过滤出包含搜索字符串的行
        .collect() // 收集结果并返回
}

// 不区分大小写的搜索函数，返回匹配行的引用
pub fn search_case_insensitive<'a>(search_str: String, content: &'a str) -> Vec<&'a str> {
    content
        .lines() 
        .filter(|line| line.to_lowercase().contains(&search_str.to_lowercase())) // 不区分大小写的过滤
        .collect() 
}

// 测试模块
#[cfg(test)]
mod tests {
    use super::*; // 导入外部模块

    // 测试区分大小写的搜索函数
    #[test]
    fn test_search() {
        let search_str = String::from("safe"); // 设置搜索字符串
        let content = String::from("\
Rust:
safe, fast, productive.
Pick three."); // 设置测试内容

        // 调用 search 函数并验证返回值
        assert_eq!(vec!["safe, fast, productive."], search(search_str, &content));
    }

    // 测试不区分大小写的搜索函数
    #[test]
    fn test_search_case_insensitive() {
        let search_str = String::from("rUst"); // 设置搜索字符串
        let content = String::from("\
Rust:
safe, fast, productive.
Pick three."); // 设置测试内容

        // 调用 search_case_insensitive 函数并验证返回值
        assert_eq!(vec!["Rust:"], search_case_insensitive(search_str, &content));
    }
}

  • Config 结构体：

    • 用于存储程序的配置，包括搜索的字符串、文件名和是否忽略大小写的标志。

    Config::new 方法：

    • 从命令行参数中读取配置，并返回一个 Config 实例。
    • 如果没有提供必要的命令行参数，会返回错误。
    • RIPGREP_CASE_INSENSITIVE 环境变量用于决定是否进行不区分大小写的搜索。

    run 函数：

    • 读取指定文件的内容并进行搜索。
    • 根据 case_insensitive 字段的值选择执行区分大小写或不区分大小写的搜索。

    search 函数：

    • 该函数接受搜索字符串和文件内容，并返回一个包含匹配行的向量。
    • 使用 lines 方法将内容按行拆分，然后通过 filter 过滤出包含搜索字符串的行。

    search_case_insensitive 函数：

    • 该函数与 search 类似，但会将搜索字符串和文件内容都转换为小写，以实现不区分大小写的搜索。

    测试模块：

    • 使用 #[cfg(test)] 标记模块，表示它是测试代码。
    • 包含两个测试函数：test_search 和 test_search_case_insensitive，分别用于测试区分大小写和不区分大小写的搜索功能。

main.rs 代码

// main.rs
use std::env;
use std::process;
use ripgrep::Config;

fn main() {
    let config:Config = Config::new(env::args()).unwrap_or_else(|err|{
        println!("Problem parsing config: {}",err);
        process::exit(1);
    });

    println!("search_str is: {}", &config.search_str);  // 打印搜索字符串
    println!("file_name is: {}", &config.file_name);    // 打印文件名

    if let Err(e) = ripgrep::run(config) {
        eprintln!("Application Error: {}", e);  
        process::exit(1);  // 错误时退出
    }
}
  • 命令行参数：通过 env::args() 获取命令行输入的参数。
  • 错误处理：通过 unwrap_or_else 来处理配置解析中的错误，若出现错误则输出提示并退出程序。
  • 运行搜索：调用 ripgrep::run() 执行文件搜索任务。
